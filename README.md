# 🌌 3D Character Carousel Slider

A cinematic 3D rotating image carousel built using pure HTML and CSS.

This project creates an orbit-style slideshow effect where image cards rotate around a central character for a modern game-launcher inspired UI.

---

## ✨ Features

- Pure CSS 3D image carousel
- Orbit-style rotating slideshow
- Perspective depth animation
- Responsive design
- Modern typography
- Cinematic layered layout
- Character-centered UI
- No JavaScript required

---

## 📸 Preview

- 3D rotating image ring
- Character orbit effect
- Animated depth perspective
- Modern game-style interface
